# 西阵订单 PHP SaaS 项目状态

更新时间：2026-06-19

## 项目定位

`php-saas/` 是当前新订单系统的 PHP SaaS 重构项目，用于替代原 Rust 主开发方向。旧系统 `/old` 继续作为线上业务系统和功能参考，不在新开发中直接修改。`rust-backup/` 作为原 Rust 版本备份保留。

当前开发策略：

- 先用原生 PHP + JSON 数据源把页面、流程、权限边界跑通。
- 同步保留 MySQL 存储接口和迁移 SQL，后续服务器启用 `pdo_mysql` 后可切换到主库 + 租户独立库。
- 页面展示主要参考 `sample.html` 和 `sample-admin.html`，订单表格保持密集后台操作风格。
- 租户侧已启用权限严格拦截：路由、操作、店铺范围、订单范围、图片上传/删除、员工/店铺/设置/导入导出等入口均按员工权限判断。
- SaaS 超管已启用租户级功能授权：平台、订单、物流、利润统计、导入导出、邮件、图片、店铺/员工/设置等功能必须由超管开通后租户侧才展示和访问。
- SaaS 超管已接管平台级全局设置：物流编号对照表、ShowAPI 配置、轮循代理不由租户侧维护；租户系统设置中的公司资料、订单参数、国内快递设置、1688 接口配置、利润计算设置全部为当前租户独立配置。
- 旧 `/old/setting.ini`、`/old/setting.php` 和旧 1688 配置只作为超管迁移参考，不再作为租户设置默认值来源；1688 配置正文只读取当前租户专属 `storage/tenants/{tenant}/config/1688/apikeys.conf`。
